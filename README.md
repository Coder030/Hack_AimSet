## Getting Started

First, go to your CLI and type in - 
#### git clone https://github.com/Coder030/Hack_AimSet  

Then run the development server using:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
